# 遊戲選單

這個專案包含兩款簡單的網頁遊戲：

- 踩地雷
- 小蜜蜂抓抓樂

## 使用方法

1. 開啟 `index.html`。
2. 在上方遊戲選單中選擇要玩的遊戲。
3. 踩地雷：選擇棋盤大小與地雷數量，然後點擊網格格子進行遊戲。
4. 小蜜蜂抓抓樂：點擊「開始小蜜蜂遊戲」，在時間內點擊藏在格子裡的小蜜蜂。

## 遊戲規則

### 踩地雷
- 左鍵點擊開啟格子。
- 右鍵點擊插旗標記地雷。
- 空白格會自動展開相鄰空格。
- 踩到地雷遊戲結束。
- 打開所有非地雷格則勝利。

### 小蜜蜂抓抓樂
- 點擊「開始小蜜蜂遊戲」啟動。
- 每波有多隻小蜜蜂會出現在不同格子裡。
- 在倒數計時結束前，點擊藏著小蜜蜂的格子即可得分。
- 達到目標分數即可獲勝。
