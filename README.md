# 遊戲選單

這個專案包含兩款簡單的網頁遊戲：

- 踩地雷
- Galaxian 小蜜蜂

## 使用方法

1. 開啟 `index.html`。
2. 在上方遊戲選單中選擇要玩的遊戲。
3. 踩地雷：選擇棋盤大小與地雷數量，然後點擊網格格子進行遊戲。
4. Galaxian 小蜜蜂：點擊「開始 Galaxian」，使用鍵盤控制飛船射擊敵人。

## 遊戲規則

### 踩地雷
- 左鍵點擊開啟格子。
- 右鍵點擊插旗標記地雷。
- 空白格會自動展開相鄰空格。
- 踩到地雷遊戲結束。
- 打開所有非地雷格則勝利。

### Galaxian 小蜜蜂
- 點擊「開始 Galaxian」啟動。
- 使用左右箭頭鍵移動飛船。
- 按空白鍵射擊子彈擊中敵人。
- 敵人會左右移動並下降。
- 擊中所有敵人獲勝，被敵人擊中或敵人到達底部則失敗。
